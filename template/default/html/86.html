智能制造新机遇，卓信创驰E521大展身手
2021-01-26

随着全球各国的智能制造政策进入深水区 诸如工业4.0、工业互联网、中国制造2025等计划不断落地开花，智能制造行业利好的消息也不断传来，加上2020年全球受疫情的冲击的影响，整个制造业，受限于人的因素越来越多，受益于智能的因素也越来越多，市场需求的不断增加,加速推进了制造业的智能化转型，边缘计算作为智能制造的关键环节也变得越来越受到重视。


利用边缘计算可以让工厂运营人员实时感知环境风险、实时监控生产过程、实时展示运营数据、减少生产过程中的风险、提高生产效率、优化制造成本。

卓信创驰E521边缘计算的利器算的利器

E521 高扩展性无风扇工业计算机 采用Intel® 6/7/8/9代桌面CPU，具备一个 1 x VGA接口 和 2 x DP接口，可支持双4K独立显示，有 3 个 Intel GbE LAN，其中两个 Intel i211-AT GbE LAN，一个 Intel i219-LM GbE LAN，支持TSN时间敏感网络，可同时满足通讯和数据采集要求，6个数据串口，包括5 个 RS232 和 1 个 RS485，可以满足工业制造现场的多种串行通信需求；总计8个USB接口， 包含4 个 USB3.0 和 4 个 USB2.0 接口，可以连接多种USB设备，大大增强了主机的可扩展性，1 x USIM可用于3G/4G LTE通信，实现便捷的无线网络连接， 1 x mSATA 接口支持市面上大部分主流的mSATA硬盘，最多3 个 SATA3.0接口可以实现磁盘阵列并轻松实现大容量存储。 1 个 PICe3.0 x16、 1个PCIe2.0 x2 搭配1个全尺寸Mini-PCIe接口，方便扩展更多外接设备，加固级无风扇结构设计 ，大幅提升设备稳定性和使用寿命。E521还附带了壁挂安装套件，让设备的安装更加简单便捷。

E521在智能制造现场应用

卓信创驰的客户可采用IPC帮助智能用户快速搭建靠近工厂物联网设备数据源头的边缘计算平台，进行实时的数据采集分析，建立虚拟数据模型，及时感知生产过程中的风险和环境变化，对设备耗能需求进行优化，提高生产效率，降低生产成本。工厂用户利用传感器、工业相机和端侧IPC进行本地数据采集，通过边缘计算智能网关进行数据交互，在边缘侧IPC进行数据过滤、清洗等实时处理，同时边缘计算平台运行在TSN时间敏感网络上，保证了数据交互的即时性。用户的工厂还可以尝试利用虚拟化技术软件建模实现工业控制器，通过边缘侧平台发布指令对产线现场进行控制。

 

典型范例是卓信创驰的客户利用工业相机、端侧数据采集设备、边缘侧计算设备E521等组合为苏州某工厂量身定制了一套边缘计算平台，工业智能相机采集到现场的数据后，通过端侧数据采集设备集成的具备TSN时间敏感网络功能的数据采集卡实时将数据传输给边缘计算网关，经过网关再传输给E521进行边缘侧的数据分析和计算，并返回指令给现场设备进行控制和操作。

依靠这套边缘计算平台，这家工厂用户可以根据边缘计算实时分析数据的结果，使制造流程能更快速的做出相应和变动，如在机器过热前自动断电、更加合理的分配机台和对焊接品质的预处理等，大幅降低了生产过程中的隐患和制造成本。
