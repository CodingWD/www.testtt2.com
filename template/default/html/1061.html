助力机器视觉领域发展 │ 卓信创驰亮相2023机器视觉技术与应用
2023-05-26

5月11日，由高工机器人、高工机器人产业研究所（GGII）主办的2023机器视觉技术与应用峰会在深圳机场凯悦酒店成功举办。本次峰会以“视界无疆”为主题，吸引了400+产业链上下游应用企业领袖、行业专家出席。


卓信创驰作为机器视觉领域的设备提供商，受邀参与到此次峰会，同时携V510、E550、E612系列视觉计算机等产品亮相。



我们知道，应用于机器视觉领域的工业计算机，不仅需要有强大的CPU性能支撑，还需要满足在严苛的工业环境中连接多个相机的需求，同时POE功能的相机在其中的应用也越来越广泛，已成为行业的大趋势。在卓信创驰此次参与亮相的产品中，E612/E550/V510系列工业视觉计算机都具有5个以上的网口。以E612系列视觉计算机为例，基于Intel第11代Tiger Lake-U系列CPU平台，具有强大的性能，可以在严苛的工业环境中稳定地运行，6个网口其中4个网口带POE功能的设计，完美符合了现在机器视觉领域的应用趋势，引起了现场的高度关注。


E612系列嵌入式视觉计算机


  • Intel 第11代Tiger Lake-U系列处理器

  • 最高64GB 双通道DDR4 3200MT/s SO-DIMM内存

  • 1 x DP, 1 x HDMI, 支持双独立显示

  • 6 x Intel i225-V / i226-V GbE LAN（其中4个支持POE）

  • 3 x RS-232, 3 x RS-232/422/485

  • 4 x USB 3.0

  • 坚固紧凑设计



E550系列嵌入式视觉计算机


  • Intel 第6~9代台式机处理器（LGA1151）

  • 最高32GB 双通道DDR4 SO-DIMM内存

  • 1 x DP, 1 x VGA, 支持双独立显示

  • 6 x GbE LAN (5 x Intel i210-AT/i211-AT,1 x Intel i219-LM)

  • 1 x RS-232, 1 x RS-232/485

  • 16 x DI, 16 x DO

  • 4 x USB 3.0, 2 x USB 2.0

  • 1 x PCIe Gen3 x16, 1 x PCIe Gen2 x2/PCI插槽

  • 坚固紧凑设计


V510系列嵌入式视觉计算机


  • Intel 第6~9代台式机处理器（LGA1151）

  • 最高32GB 双通道DDR4 SO-DIMM内存

  • 1 x DP, 1 x VGA, 支持双独立显示

  • 5 x GbE LAN (4 x Intel i210-AT/i211-AT,1 x Intel i219-LM)

  • 1 x RS-232, 1 x RS-485

  • 8 x DI, 16 x DO

  • 6 x 光源控制，6 x 触发

  • 4 x USB 3.0, 2 x USB 2.0

  • 坚固紧凑设计



扩展卡系列(C100/C110/C200)


  • PCIe Gen2.0 x4 插槽

  • 4 x GbE LAN (C100&C110), 4 x USB 3.1(C200)

  • 支持4 POE(C100)

  • 每个网口由独立Intel i210-AT/i211-AT芯片控制（C100&C110）

  • 4 端口USB3.1，由2个独立控制器提供(C200)

  • 支持Windows 10 和 Linux系统


未来，卓信创驰将继续以技术创新为基础，聚焦于机器视觉领域，推出更多满足行业需求的产品，助力客户智能制造，为产业发展贡献一份力。
